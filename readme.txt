== Description ==

A simple plugin for a widget that lets you display a number of posts from a selected category in widget areas. 
This widget includes several options for selecting a category to display, providing a title, choosing a number of 
posts to display and featured image support.


== Installation ==

1. Upload the simple-categories-widget.zip using WordPress plugins uploader.
2. Activate the plugin
3. Under Appearance/Widgets drag and drop the Simple Categories Widget to the desired widget area
4. Fill in the options
5. Enjoy!
